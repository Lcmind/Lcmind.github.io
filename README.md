- **기술**: Flutter, FastAPI, MySQL, Firebase, Nginx
- **역할**: 백엔드 개발 100%, 프론트엔드 70%

### 2. Focus Hub
- **타입**: PWA Application
- **설명**: 초소규모 스터디/모임 생산성 도구
- **기술**: React, Supabase, Firebase, Vercel
- **역할**: 기획부터 배포까지 100% 독립 개발

### 3. 후딱가게 (Hooddak)
- **타입**: Mobile Application
- **설명**: 대학교 근처 맛집 추천 및 혼잡도 투표 앱
- **기술**: Flutter, Flask, Nginx, AWS, Firebase
- **역할**: 백엔드 100%, 프론트엔드 70%

### 4. 주인나죽어
- **타입**: PC Game
- **설명**: Unity 식물 육성 및 도감 수집 게임
- **기술**: Unity
- **역할**: 기획 50%, 에셋 구성 70%, 픽셀 아트 40%

### 5. 조인택시 (JoinTaxi)
- **타입**: Website
- **설명**: 대학생을 위한 택시 쉐어링 웹사이트
- **기술**: PHP, MySQL
- **역할**: 프론트엔드 75%, 데이터베이스 50%, Heroku 배포 100%

### 6. Posture Piece
- **타입**: Smart Watch Application 기획
- **설명**: 스마트워치 기반 자세 교정 앱 (142페이지 기획서)
- **역할**: Payment & Subscription, Compatibility, Security (100%)
